# OverDriveAuthoring
An Authoring Tool for OverDrive, a mobile rhythm game

### Installation
Uses Visual Studio 2017, C# Windows Form application.
To install:
- Clone the repo
- Open the Visual Studio Solution (.sln)
- Run the program from the bin folder, or build from the IDE
